# MESA-Stellar-Evolution-Simulations
This project investigates: Evolution of Stars of varied stellar masses from PMS to TAMS, Convective core evolution in high-mass and Low-mass stars, Effects of convective overshooting in Rotating stars- comparison with high rotators, Nuclear reaction networks and deuterium burning. Simulations were performed using MESA Ver. r10000, analyzed with Python and MESA's native mesa_reader library.
